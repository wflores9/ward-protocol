# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities to:
**security@wardprotocol.org**

Do not open public GitHub issues for security vulnerabilities.
We will acknowledge within 48 hours and provide a resolution
timeline within 5 business days.

## Scope

- ward/ Python library
- api.wardprotocol.org endpoints
- Nine-check default validation logic
- ward_signed = False invariant
- TypeScript SDK (@wardprotocol/sdk)
