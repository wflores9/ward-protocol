PREIMAGE-SHA-256 conditioned escrow. 48-hour dispute window. Ward never sees the preimage — only condition_hex. Once EscrowCreate is on-chain, Ward's operational status is permanently irrelevant.

Deterministic by design

Same inputs always produce same outcome. Institutions can audit the exact logic before deploying capital. No governance vote changes it after the fact.

Replay protection built in

Policy NFT burns atomically with EscrowFinish. A burned NFT fails Check 01 immediately. Double-claims are structurally impossible — not just policy-prohibited.

ward_signed = False # always # All 9 pass → unsigned EscrowCreate # Any failure → ClaimRejectedError(reason, check_n) institution.sign(escrow_tx) xrpl.settle() # ledger enforces
